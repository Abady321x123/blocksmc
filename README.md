# BlocksMC 
A simple scraper & wrapper for BlocksMC!

## Instalation
```
npm i blocksmc
```

## Example Usage
```js
const blocksmc = require('blocksmc');

blocksmc.player('iAbady').then(player => {
    console.log(player.game.map(g => `${g.game} ${g.stats.Deaths}`))
});

blocksmc.top('sky-wars').then(leader => {
    console.log(leader);
})
```
